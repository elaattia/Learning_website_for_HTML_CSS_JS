HTML (Hypertext Markup Language) is a language used to create web pages. It provides a structure to your content by using elements and attributes.

To create an HTML document, you need to use a text editor. Open a new document and save it with the .html extension. The basic structure of an HTML document includes the following tags:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
</body>
</html>

The head section of an HTML document contains metadata such as the title, author, and description of the web page. The title tag is used to define the title of the page that appears in the browser tab.

<head>
<title>Page Title</title>
<meta name="description" content="Description of the web page">
</head>

The body section of an HTML document contains the content that is displayed on the web page. You can add text, images, links, and other elements to the body section.

<body>
<h1>Heading 1</h1>
<p>Paragraph</p>
<img src="image.jpg" alt="Image">
<a href="https://www.example.com/">Link</a>
</body>

HTML tags are used to define the structure and content of the web page. Here are some of the most commonly used HTML tags:

  • <h1>, <h2>, <h3>, <h4>, <h5>, <h6> - Used to define headings.
  • <p> - Used to define paragraphs.
  • <a> - Used to define hyperlinks.
  • <img> - Used to display images.
  • <ul> - Used to create an unordered list.
  • <ol> - Used to create an ordered list.
  • <li> - Used to define list items.
  • <table> - Used to create tables.
  • <tr> - Used to define table rows.
  • <td> - Used to define table data.

Attributes provide additional information about HTML elements. They are defined within the opening tag of an element. Here are some of the most commonly used attributes:

  • 'class' - Used to define a class for an element.
  • 'id' - Used to define a unique identifier for an element.
  • 'src' - Used to define the source of an image or media file.
  • 'href' - Used to define the URL of a hyperlink.
  • 'alt' - Used to define alternative text for an image.
  • 'style' - Used to define inline styles for an HTML element. This attribute is commonly used to apply CSS styles to a specific element.
  • 'title' - Used to define a title for an HTML element. This attribute is commonly used to provide additional information about an element when the user hovers over it.

    • Here's an example of using these attributes together:


    <a href="https://www.example.com" class="button" id="example-link" title="Visit Example">Example Website</a>
    <img src="example-image.jpg" alt="Example Image" style="width: 100%;">

    Comments can be added to an HTML document to provide notes for the developer. They are not visible on the web page. Comments are defined using the <!-- --> tag.

    <!-- This is a comment -->

    HTML forms are used to collect information from users. The most commonly used form elements are:

  • <form> - Used to create a form.
  • <input> - Used to create input fields.
  • <textarea> - Used to create a text area.
  • <select> - Used to create a dropdown list.
  • <option> - Used to define options for a dropdown list.
  • <button> - Used to create a button.
  • <label> - Used to provide a label for an input field. The 'for' attribute of the label element should match the 'id' attribute of the input field.
  • 'type' - Used to define the type of input field. Common values include 'text', 'email', 'password', 'checkbox', 'radio', 'submit', and 'reset'.
  • 'name' - Used to define the name of the input field. This attribute is used to identify the field when the form is submitted.
  • 'value' - Used to define the value of the input field. This attribute is used to provide a default value for the field.
  • 'required' - Used to make an input field required. If this attribute is present, the user must enter a value in the field before the form can be submitted.

    • Here is an example of an HTML form:


    <form>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required>
    <br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <br>
    <label for="password">Password:</label>
    <input type="password" id="password" name="password" required>
    <br>
    <label for="gender">Gender:</label>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label>
    <br> <label for="country">Country:</label>
    <select id="country" name="country">
    <option value="usa">USA</option>
    <option value="uk">UK</option>
    <option value="canada">Canada</option>
    </select>
    <br>
    <button type="submit">Submit</button>
    </form>

    HTML5 introduced semantic elements that provide a meaning to the content of the web page. These elements make it easier for search engines and screen readers to understand the content. Here are some of the most commonly used semantic elements:

  • <header> - Used to define the header of a web page or section.
  • <nav> - Used to define the navigation section of a web page.
  • <main> - Used to define the main content section of a web page.
  • <section> - Used to define a section of the web page.
  • <article> - Used to define an article or blog post.
  • <aside> - Used to define a side section of the web page.
  • <footer> - Used to define the footer of a web page or section.

    • HTML5 provides built-in support for playing audio and video files. You can use the

    <audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
    </audio>
    <video controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video element.
    </video>

    Meta tags provide information about the web page that is used by search engines and social media platforms. Here are some of the most commonly used meta tags:

  • <meta name="description" content="Description of the web page"> - Used to define the description of the web page.
  • <meta name="keywords" content="Keywords"> - Used to define the keywords of the web page.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> - Used to define the viewport of the web page for responsive design.
