Introduction to JavaScript


JavaScript is a programming language that is commonly used in web development to add dynamic functionality to websites. It was created in 1995 by Brendan Eich, and it is now one of the most popular programming languages in the world.


JavaScript is a high-level, interpreted language, which means that it is designed to be easy for humans to read and write, and it is executed on-the-fly by the browser or other JavaScript engines, without needing to be compiled first.


Setting up the Environment


Before we start learning JavaScript, we need to set up our environment. We will need a text editor to write our code in and a browser to test our code in.


Text Editors

There are many text editors to choose from, including Visual Studio Code, Sublime Text, Atom, and Notepad++. Choose one that you like and install it on your computer.


Browsers

There are also many browsers to choose from, including Google Chrome, Firefox, Safari, and Microsoft Edge. Choose one that you like and install it on your computer.

In JavaScript, we use variables to store data. We can declare a variable using the let or const keyword, followed by the variable name and an optional initial value.

let x = 10;
                    const y = "Hello, world!";

JavaScript has several data types, including numbers, strings, booleans, arrays, objects, and null/undefined.

let num = 10;
                    let str = "Hello, world!";
                    let bool = true;
                    let arr = [1, 2, 3];
                    let obj = { name: "John", age: 30 };
                    let n = null;
                    let u = undefined;

JavaScript has several operators, including arithmetic, comparison, logical, and assignment operators.

let x = 10 + 5; // 15
                        let y = "Hello" + "world"; // "Helloworld"
                        let z = 10 === 5; // false
                        let a = true && false; // false
                        let b = 5; b += 2; // b is now 7

JavaScript has conditional statements that allow us to execute code based on a condition. The most common conditional statement is the if statement.

let x = 10;

                    if (x > 5) {
                      console.log("x is greater than 5");
                    } else {
                      console.log("x is less than or equal to 5");
                    }

JavaScript has loops that allow us to execute code repeatedly. The most common loops are the for and while loops.

for (let i = 0; i < 10; i++) {
                        console.log(i);
                      }
                      
                      let i = 0;
                      while (i < 10) {
                        console.log(i);
                        i++;
                      }

JavaScript has functions that allow us to reuse code. We can declare a function using the function keyword, followed by the function name, optional parameters, and the function body.

function sayHello(name) {
                        console.log("Hello, " + name + "!");
                      }
                      
                      sayHello("John"); // "Hello, John!"
